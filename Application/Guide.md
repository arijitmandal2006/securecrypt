* * * * *

#### üîí How to Encrypt a File

1.  Type `1` and press **Enter** (Select "Encrypt").

2.  Type `2` and press **Enter** (Select "File Input").

3.  A **window will pop up**. Find the file you want to lock (like a photo, PDF, or text file) and click **Open**.

4.  The app will ask for a **Password**. Type a secret password and press **Enter**.

    -   *Note: Maybe you won't see the password while typing (for security), but it is being typed!*

5.  **Done!** A new file ending in `.enc` (e.g., `photo.png.enc`) is created in the same folder. You can now delete the original file if you want.

* * * * *

#### üîì How to Decrypt (Unlock) a File

1.  Type `2` and press **Enter** (Select "Decrypt").

2.  Type `2` and press **Enter** (Select "File Input").

3.  Select the **`.enc` file** you want to unlock.

4.  Type the **same password** you used to lock it.

5.  **Done!** Your original file will reappear (ending in `.dec`), ready to use.
